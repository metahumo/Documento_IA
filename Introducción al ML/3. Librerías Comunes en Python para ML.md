# Librerías Comunes en Python para Machine Learning

En este documento revisamos las principales librerías utilizadas en proyectos de *machine learning* con Python. Para cada una veremos:

- Instalación
- Para qué sirve
- Un ejemplo básico
- Un ejemplo aplicado a ciberseguridad

Las librerías incluidas son: **Pandas, NumPy, Scikit-learn, Matplotlib/Seaborn y TensorFlow**.

---

# 1. Pandas

## 1.1 Instalación

```bash
pip install pandas
````

## 1.2 ¿Para qué sirve?

Pandas es una librería orientada al tratamiento de datos. Permite:

- Cargar datos desde múltiples formatos
    
- Limpieza y transformación
    
- Selección y filtrado
    
- Análisis estadístico básico
    

Es fundamental para preparar datasets antes de entrenar un modelo.

## 1.3 Ejemplo básico

```python
import pandas as pd

df = pd.DataFrame({
    "edad": [21, 23, 30],
    "ingresos": [1500, 2000, 3000]
})

print(df.describe())
```

## 1.4 Ejemplo aplicado a ciberseguridad

```python
import pandas as pd

logs = pd.read_csv("traffic.csv")

# Filtrar solo conexiones hacia el puerto 22 (SSH)
ssh = logs[logs["dest_port"] == 22]

print(ssh.head())
```

Este tipo de filtrado es útil para analizar patrones de acceso o posibles ataques de fuerza bruta.

# 2. NumPy

## 2.1 Instalación

```bash
pip install numpy
```

## 2.2 ¿Para qué sirve?

NumPy facilita operaciones:

- Con vectores y matrices
    
- Álgebra lineal
    
- Cómputo numérico eficiente
    

Muchos frameworks de ML lo usan internamente.

## 2.3 Ejemplo básico

```python
import numpy as np

a = np.array([1, 2, 3])
print(a * 2)
```

## 2.4 Ejemplo aplicado a ciberseguridad

```python
import numpy as np

# Entropía aproximada de bytes en un archivo binario
data = np.random.randint(0, 256, 1000)
hist = np.bincount(data) / len(data)
entropy = -np.sum(hist * np.log2(hist + 1e-9))

print(entropy)
```

La entropía es un indicador útil para detectar malware ofuscado.

# 3. Scikit-learn

## 3.1 Instalación

```bash
pip install scikit-learn
```

## 3.2 ¿Para qué sirve?

Es una librería clásica para ML con:

- Modelos supervisados (regresión, clasificación)
    
- Modelos no supervisados (clustering)
    
- Preprocesamiento
    
- Divisiones del dataset
    

Ideal para prototipos y proyectos rápidos.

## 3.3 Ejemplo básico (Clasificación)

```python
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier

X = [[1], [2], [3], [4]]
y = [0, 0, 1, 1]

model = DecisionTreeClassifier()
model.fit(X, y)

print(model.predict([[2.5]]))
```

## 3.4 Ejemplo aplicado a ciberseguridad (Detección de malware)

```python
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier

# Características: tamaño, entropía, nº imports
X = [[200, 3.1, 10], [400, 7.4, 30], [150, 2.0, 5]]
y = [0, 1, 0]  # 1=malicioso

model = RandomForestClassifier()
model.fit(X, y)

print(model.predict([[350, 6.0, 20]]))
```

# 4. Matplotlib y Seaborn

## 4.1 Instalación

```bash
pip install matplotlib 
pip install seaborn
```

## 4.2 ¿Para qué sirven?

Son librerías de visualización que ayudan a:

- Explorar datos
    
- Mostrar distribuciones
    
- Detectar patrones
    

Matplotlib es más general.  
Seaborn es más estética y estadística.

## 4.3 Ejemplo básico

```python
import matplotlib.pyplot as plt
import seaborn as sns

data = [1,2,3,4,5]

sns.histplot(data)
plt.show()
```

## 4.4 Ejemplo aplicado a ciberseguridad

```python
import seaborn as sns
import matplotlib.pyplot as plt

connections = [30,10,5,100,3,60,15]

sns.boxplot(connections)
plt.title("Distribución de conexiones por host")
plt.show()
```

Visualizar eventos ayuda a detectar outliers como picos de tráfico sospechoso.

# 5. TensorFlow

## 5.1 Instalación

```bash
pip install tensorflow
```

## 5.2 ¿Para qué sirve?

TensorFlow es un framework para deep learning que permite:

- Redes neuronales
    
- Modelos complejos
    
- GPU/TPU
    
- Producción y despliegue
    

Se usa para tareas avanzadas de ML.

## 5.3 Ejemplo básico

```python
import tensorflow as tf

model = tf.keras.Sequential([
    tf.keras.layers.Dense(1, input_shape=(1,))
])

model.compile(optimizer="sgd", loss="mse")
model.fit([1,2,3,4], [2,4,6,8], epochs=10)
```

## 5.4 Ejemplo aplicado a ciberseguridad (detección de anomalías)

```python
import tensorflow as tf
import numpy as np

# Datos simulados de tráfico normal
normal = np.random.normal(0, 1, (100, 10))

model = tf.keras.Sequential([
    tf.keras.layers.Dense(5, activation="relu"),
    tf.keras.layers.Dense(1)
])

model.compile(optimizer="adam", loss="mse")
model.fit(normal, np.zeros((100, 1)), epochs=5)

# Entrada sospechosa
suspicious = np.random.normal(5, 1, (1,10))
score = model.predict(suspicious)

print("Anomaly Score:", score)
```

Este tipo de red puede utilizarse para detectar comportamientos atípicos.
