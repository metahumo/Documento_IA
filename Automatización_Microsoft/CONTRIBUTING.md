# Contribuir a este repositorio

¡Gracias por tu interés en contribuir! Este repositorio busca ser un recurso educativo sobre automatización con Microsoft Power Platform.

## Cómo contribuir

### Reportar errores o sugerencias

Si encuentras errores, información desactualizada o tienes sugerencias:

1. Abre un **Issue** describiendo el problema o la mejora
2. Incluye capturas de pantalla si es relevante
3. Especifica qué documento o sección necesita actualización

### Añadir contenido

Si quieres aportar nuevos casos prácticos o guías:

1. **Fork** el repositorio
2. Crea una rama con un nombre descriptivo: `git checkout -b feature/nuevo-caso-practico`
3. Añade tu contenido siguiendo la estructura existente:
   - Usa Markdown para la documentación
   - Incluye capturas de pantalla en la carpeta `Images/`
   - Mantén un tono educativo y accesible
4. Asegúrate de que tu contenido:
   - Es preciso y está probado
   - Incluye buenas prácticas de seguridad
   - Respeta las políticas corporativas de Microsoft 365
5. Envía un **Pull Request** con una descripción clara de los cambios

## Estilo y formato

- Usa Markdown estándar
- Incluye títulos descriptivos con `#`, `##`, `###`
- Las capturas deben ser claras y estar en formato PNG o JPG
- Menciona versiones específicas cuando sea relevante (ej: "Outlook V3")
- Revisa la ortografía y gramática antes de enviar

## Código de conducta

- Sé respetuoso/a con otros colaboradores
- Acepta feedback constructivo
- Enfoca las contribuciones en el valor educativo
- No compartas información confidencial o credenciales reales

## Licencia

Al contribuir, aceptas que tu contenido se publique bajo la licencia MIT de este repositorio.
