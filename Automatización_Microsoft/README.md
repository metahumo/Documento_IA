# Automatización en Microsoft 365

Repositorio de documentación y guías prácticas sobre automatización de procesos utilizando las herramientas de Microsoft Power Platform, enfocado en optimizar tareas repetitivas y aumentar la productividad en entornos corporativos.

## Contenido

Este repositorio contiene documentación sobre:

- **Introducción a Power Platform**: Conceptos fundamentales, herramientas disponibles y buenas prácticas de seguridad
- **Power Automate**: Guías detalladas y casos prácticos de automatización de flujos de trabajo

## Objetivo

El objetivo principal es proporcionar recursos prácticos que permitan:

- Automatizar tareas repetitivas que consumen tiempo
- Reducir el factor de error humano en procesos manuales
- Incrementar la productividad individual y de equipos
- Facilitar la integración entre diferentes aplicaciones de Microsoft 365

## ¿Qué es Power Platform?

**Power Platform** es un conjunto de herramientas de Microsoft que permite crear soluciones empresariales y automatizar procesos sin necesidad de programación avanzada. Incluye:

- **Power Automate**: Automatización de flujos de trabajo entre aplicaciones
- **Power Apps**: Creación de aplicaciones personalizadas
- **Power BI**: Análisis y visualización de datos

## Conceptos clave

- **Trigger / Disparador**: Evento que inicia un flujo o acción automatizada
- **Action / Acción**: Tarea que se realiza una vez activado un trigger
- **Conectores**: Elementos que integran diferentes aplicaciones y servicios
- **Canvas**: Entorno de diseño visual en Power Apps
- **Azure**: Plataforma en la nube que proporciona infraestructura segura

## Ciberseguridad

Es fundamental seguir buenas prácticas de seguridad:

-  Usar siempre correo corporativo y cuentas autorizadas
-  Evitar compartir información sensible con herramientas externas no autorizadas
-  Revisar permisos y conectores utilizados en cada flujo
-  Documentar y revisar periódicamente los flujos activos

## Ejemplos de uso

Algunas automatizaciones útiles que puedes implementar:

- Enviar correos o mensajes en Teams automáticamente
- Avisar sobre formaciones obligatorias pendientes
- Recordatorios automáticos sobre vencimiento de documentación
- Notificaciones cuando se generan nuevos registros
- Clasificación y archivo automático de documentos adjuntos
- Creación de informes automáticos de actividades
